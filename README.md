# TextAnalyzer

A front end task I made to analyze text for specific details, including:
* Word count
* Number of letters
* Longest word
* Average word length
* Reading duration in seconds
* Median word length
* Median word length when sorted by length
* Top 5 most common words
* Text language (English and Turkish only)
